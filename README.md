💸 Expense Tracker

A modern and responsive Expense Tracker Web App built using HTML, CSS, and JavaScript.
It helps users manage daily finances with features like multi-language support, theme switching, and a basic audit system.

---

🚀 Features

- ➕ Add income and expenses
- 📊 Real-time balance, income & expense summary
- ❌ Delete transactions
- 🌐 Multi-language support (switch between languages)
- 🌙☀️ Light & Dark Mode toggle
- 🔍 Basic audit system (track user actions like add/delete)
- 💾 Data stored using Local Storage
- 📱 Fully responsive design

---

🛠️ Tech Stack

- HTML – Structure
- CSS – Styling & Themes
- JavaScript – Logic & Interactivity
- Local Storage – Data persistence

⚙️ How It Works

1. Enter transaction title
2. Enter amount
   - Positive → Income
   - Negative → Expense
3. Add transaction
4. Switch between 🌐 languages or 🌙☀️ themes anytime
5. Track actions in the audit log
